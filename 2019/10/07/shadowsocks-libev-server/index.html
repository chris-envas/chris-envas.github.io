本文主要介绍如何在ubuntu下部署ss服务，帮助自己更好的查阅资料提升自己，ss服务有相当多的开源脚本，本次使用的服务是shadowsocks-libev

shadowsocks-libev on ubuntu

如果显示进程占用，查看 apt 的进程是否在运行
while true;do ps aux | grep apt ;sleep 4 ; clear; done

  • Update System packages cache
    apt-get update
  • 安装 依赖开发库和开发环境
    apt-get install build-essential libsodium-dev libc-ares-dev libmbedtls-dev libpcre3-dev libev-dev
  • 编译安装
  • 建立ss-server配置文件
  • 建立systemd配置文件
  • 重载服务
    systemctl daemon-reload
  • 开启服务
    systemctl restart ss-server.service
  • 检测上面定义的18443端口是否开启
    netstat -tlnp
