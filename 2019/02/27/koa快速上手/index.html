koa快速上手

发表于 2019-02-27 | 更新于 2019-07-26 | 分类于 后端

Koa 是一个新的 web 框架，由 Express 幕后的原班人马打造， 致力于成为 web 应用和 API 开发领域中的一个更小、更富有表现力、更健壮的基石。 通过利用 async 函数，Koa 帮你丢弃回调函数，并有力地增强错误处理。 Koa 并没有捆绑任何中间件， 而是提供了一套优雅的方法，帮助您快速而愉快地编写服务端应用程序。本文以实战为主，阐述了常规API及概念的用法，以及许多常用的koa类库使用
