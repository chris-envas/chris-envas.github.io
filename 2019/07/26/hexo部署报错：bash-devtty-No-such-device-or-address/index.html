发表于 2019-07-26 | 分类于 其他

原因

经过一番排查，发现似乎是我重装系统后，网络发生了一些变化，使用hexo deploy部署到github时，出现了无法找到IP地址的情况，怎么办，遂把hexo的配置文件_config做了小小的修改，不再采用https的形式上传，而是使用ssh的方式！
