polipo 把 socks5 转 http 代理

发表于 2018-12-19 | 更新于 2019-07-26 | 分类于 其他

git clone 某些资源的时候，一旦资源被墙将导致下载的速度变得特别的慢或者甚至根本无法下载，基于已有的梯子，可以为git的资源下载设置代理，突破墙的阻隔获取资源

谷歌搜索:
polipo-20080907-win32.zip

在 polipo-20080907-win32/ 中
vi config.conf

vi polipo.cmd

打开 polipo.cmd

使用示例

wget curl npm 等命令只需要声明 HTTP_PROXY 和 HTTPS_PROXY 变量，就能让这些程序自动走 http 代理，
部分程序 或更新比较勤快的程序 还支持 socks5://

之后只要在这次会话内的上述程序，都会自动走上面定义的 http 代理
