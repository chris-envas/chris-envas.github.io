Webpack配置

发表于 2018-11-27 | 更新于 2019-07-26 | 分类于 前端

本文使用webpack4.25.1
webpack是一个现代 JavaScript 应用程序的静态模块打包器(static module bundler),选择它非必要而是需要,目前热门的Vue,React的前端MVVM框架均配置webpack相关的脚手架,前端开发人员可以使用该脚手架快速的迭代开发，但在撸码的过程中，脚手架内复杂的配置却往往令人望而生畏，也许有人会问，脚手架帮你已经配置的非常完整了，为什么还要自己辛苦学习配置一遍，其实我们的目的并非去取代它，而是当我们出现问题时，可以进行快速的排查，并且去除某些我们我们不需要的模块，或者新增我们需要的模块，经历一个从无到有的过程，才可以更好的提升工作效率
