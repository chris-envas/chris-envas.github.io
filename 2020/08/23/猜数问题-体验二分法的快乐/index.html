猜数问题-体验二分法的快乐

发表于 2020-08-23 | 更新于 2020-08-26

前言

二分法数学概念：

若已知函数f(x) = 0的解，则

  • 1、确定区间[a,b]，一定包含该方程式的解
  • 2、求该区间中点m = (a+b)/2，并确定f(m)的值
  • 3、若f(m)与f(a)正负号相同号相同，去[m,b]为新的区间，反之则取[a,m]
  • 4、重复2、3步直至取得理想精度值

时间复杂度：

若区间范围为n个元素，每次查找区间为：n，n/2，n/4，…，n/2^k

且 n/2^k >= 1，可得k = log2n（以2为底，n的对数），故时间复杂度：O(n) = logn
